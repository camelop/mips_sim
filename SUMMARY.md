# MIPS-Simulator项目总结

## 项目简介

本项目为上海交通大学[2017年ACM班PPCA]("https://acm.sjtu.edu.cn/wiki/PPCA_2017)个人大作业。

项目应为一个可以正确模拟MIPS功能的解释器，在完整的项目中应包含与CPU类似的pipeline结构。

除了上述内容外，还可实现(括号中为本项目中实现的)：
- [x] 静态/动态分支预测
- [x] 并发的五级流水（多线程）
- [ ] Tomasulo算法以及对应的多级流水
- [ ] 不同的架构（如龙芯九级流水）

在此对学到的相关知识以及整个经过做一总结。
